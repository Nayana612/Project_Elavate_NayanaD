# Face Mask Detection 🔍😷

A real-time system that detects face mask usage via webcam. If a person isn’t wearing a mask, it plays a voice alert and alarm.

## Tools
- Python, OpenCV, Keras
- pyttsx3 (voice), playsound (alarm)

## Steps
1. Load model
2. Detect face
3. Predict: Mask / No Mask
4. Alert if needed

## Features
✔ Real-time webcam feed  
✔ Voice + alarm alert  
✔ Simple UI overlay
